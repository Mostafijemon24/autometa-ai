# AutoMeta AI

AutoMeta AI is a WordPress plugin that automates SEO meta titles and
descriptions using AI-driven logic.

The goal is to reduce repetitive SEO tasks while keeping full control
in the hands of the site owner.

## Key Features
- AI-generated meta titles and descriptions
- SEO-aware output structure
- Lightweight and performance-friendly
- Designed for scalable WordPress setups
